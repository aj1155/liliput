## Liliput
URL 단축 서비스  
Official Site :
<http://nexters.me>

### Liliput?
Liliput는 긴 URL을 짧은 URL로 바꾸어주는 서비스입니다.
* URL을 짧게
* 단축 URL 커스텀
* Base 62 인코딩
* RESTful API 제공
* Log를 통한 데이터 분석

### Architecture
작성중...
- JDK 8+
- Tomcat 8
- nginx
- Spring Boot
- Angular2
- JPA
